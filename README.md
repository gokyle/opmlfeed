[opmlfeed]
==========

opmlfeed is an opmlfeed subscription service.


Dependencies
------------

* redis (although you can change the database backend)


Deployment
----------
* Install redis
* Edit env.sh to suit your needs and source it
* go build && ./opmlfeed


Notes
-----

* It would be trivial to adapt it other uses.
